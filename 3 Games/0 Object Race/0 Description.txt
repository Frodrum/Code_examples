Реализуйте игру, в которой некоторое количество объектов, в автоматическом режиме  соревнуется в скоростном прохождении дистанции.

Условия:
1. Пользователь должен видеть процесс прохождения дистанции;
2. Победитель должен определяться с некоторой вероятностью и объявляться в конце;
3. Объектом может быть кто угодно (Steel ball run!);
4. Условием финиша может быть как достижение определённого показателя (Дистанция), так и окончание времени гонки (Кто впереди, тот и победил).

Пример результата:
1. Какое-то время с начала:
...🏍#1
..🚓#2
....🏃#3
....🐎#4

2. Спустя ещё какое-то время:
.........🏍#1
.....🚓#2
.........🏃#3
..........🐎#4

3. Объявление победителя:
........................................🏍#1
........................................🚓#2
............................................🏃#3
.................................................🐎#4


Winner is 🐎#4!
